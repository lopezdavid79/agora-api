Alta ▼
ReqDescripciónPrioridadEsfuerzo
#3 Endpoint para que el Gestor pueda resetear la contraseña de un candidato, generando un token de un solo uso Alta ~4h

📦 Endpoints nuevos

// Gestor+ genera token de reseteo POST /api/gestor/usuarios/:id/resetear-password Body: { } // (no requiere body, el gestor solo inicia el proceso) → 200 { "mensaje": "Token de reseteo generado", "token": "abc123" } // El candidato canjea el token (sin autenticación) POST /api/auth/resetear-password Body: { "token": "abc123", "password": "NuevaPass1", "confirmPassword": "NuevaPass1" } → 200 { "mensaje": "Contraseña actualizada correctamente" }

📦 Modelo nuevo

// src/models/TokenReseteo.js { id: INT PK, usuarioId: INT FK, token: VARCHAR(255), // crypto.randomBytes(32).toString('hex') usado: BOOLEAN default false, expiraEn: DATETIME, // ahora + 15 minutos creadoPor: INT FK // gestorId que generó el token }

📖 Flujo

Gestor → POST resetear-password → genera token de 15 min → candidato usa POST reset-password con el token → password actualizada. El token se marca como usado.

⏰ Job programado (node-cron)

npm install node-cron // src/jobs/purga-documentos.js const cron = require('node-cron'); const { Documento } = require('../models'); const fs = require('fs'); // Ejecutar cada día a las 3:00 AM cron.schedule('0 3 * * *', async () => { const treintaDias = new Date(Date.now() - 30 * 24 * 60 * 60 * 1000); const aPurgar = await Documento.findAll({ where: { deletedAt: { [Op.lte]: treintaDias } }, }); for (const doc of aPurgar) { try { fs.unlinkSync(doc.urlDrive); } catch (e) {} await doc.destroy({ force: true }); // DELETE físico } console.log(`🧹 Purga: ${aPurgar.length} documentos eliminados`); });
